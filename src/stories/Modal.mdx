A dialog overlay for focused tasks, confirmations, or detail views. Renders as a native `<dialog>` element.

## Composition

`Modal` composes `ModalHeader` and `ModalFooter` sub-components. Use them independently when building custom layouts.

## Accessibility

- Built on the native `<dialog>` element — focus is trapped inside when open.
- `Escape` key closes the modal by default.
- The dialog has `aria-labelledby` pointing to the title and `aria-describedby` pointing to the description.
- Backdrop click closes the modal when `closeOnBackdropClick` is true.
