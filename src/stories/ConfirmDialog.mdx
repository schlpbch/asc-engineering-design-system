# ConfirmDialog

A reusable `<dialog>`-based confirmation modal triggered programmatically via `window.showConfirmDialog()`. Returns a `Promise<boolean>` that resolves to `true` when confirmed, `false` when cancelled.

## Usage

Add the component once to your layout:

```astro
<ConfirmDialog />
```

Then trigger it from any script:

```js
const confirmed = await window.showConfirmDialog({
  title: 'Delete service',
  message: 'This action cannot be undone. Are you sure?',
  confirmText: 'Delete',
  cancelText: 'Cancel',
  variant: 'danger',
});

if (confirmed) {
  // perform the action
}
```

## Options

| Option | Type | Description |
|--------|------|-------------|
| `title` | `string` | Dialog heading |
| `message` | `string` | Body text |
| `confirmText` | `string` | Label for the confirm button (default: `"Confirm"`) |
| `cancelText` | `string` | Label for the cancel button (default: `"Cancel"`) |
| `variant` | `'default' \| 'danger'` | `danger` styles the confirm button destructively |

## Accessibility

- Built on the native `<dialog>` element with focus trapping.
- `Escape` key cancels and resolves the promise to `false`.
- Dangerous actions use a red confirm button to signal irreversibility.
