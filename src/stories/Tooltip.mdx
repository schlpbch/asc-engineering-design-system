# Tooltip

A small popup that reveals additional information about a trigger element on hover or focus. Use for supplementary context — never for critical information.

## Accessibility

- The tooltip panel has `role="tooltip"` and is linked to its trigger via `aria-describedby`.
- Triggered on both hover and focus so keyboard users receive the same information.
- Do not place interactive elements (links, buttons) inside a tooltip.
