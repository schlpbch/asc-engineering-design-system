# IconButton

A square button that contains only an icon, with no visible text label. Always requires an `aria-label` for accessibility.

## Accessibility

- `aria-label` is required — the button has no visible text.
- Use `title` to show a native browser tooltip on hover.
- For toggle buttons, set `aria-pressed="true"` or `"false"` to communicate state to screen readers.
