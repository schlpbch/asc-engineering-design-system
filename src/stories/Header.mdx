# Header

Top navigation bar with slots for logo, navigation links, actions, and a mobile menu. Supports sticky positioning.

## Slots

| Slot | Description |
|------|-------------|
| `logo` | Brand mark / wordmark on the left |
| `nav` | Main navigation links (hidden on mobile) |
| `actions` | Right-side icons, avatar, or CTA buttons |
| `mobile-menu` | Content shown in the mobile drawer |

## Accessibility

- Wrapped in `<header>` landmark.
- The mobile menu button has `aria-expanded` and `aria-controls` attributes.
- Navigation links should use `aria-current="page"` on the active item.
