# ProjectGrid

A grid layout for showcasing projects with name, description, tech stack tags, and an external link. Supports a featured 2-column layout and a "View all" link.

## Project data shape

```ts
type Project = {
  name: string;
  description: string;
  tech: string[];
  href?: string;
};
```

## Accessibility

- Project links use descriptive text (the project name) as the anchor label.
- Tech tags are decorative and `aria-hidden` where not meaningful as standalone labels.
