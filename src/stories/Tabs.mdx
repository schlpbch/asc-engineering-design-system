## Accessibility

- Tab list uses `role="tablist"`, each tab uses `role="tab"`, and each panel uses `role="tabpanel"`.
- Active tab has `aria-selected="true"`; panels are shown/hidden with `aria-hidden`.
- Keyboard: `ArrowLeft`/`ArrowRight` navigate between tabs; `Enter`/`Space` activates.
