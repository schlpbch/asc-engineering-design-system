# SectionHeader

A simple heading component for labeling content sections within a page. Accepts an optional icon slot to the left of the title.

## Accessibility

- Renders as `<h2>` by default — adjust heading level in context to maintain document outline.
- Icons should be `aria-hidden="true"`.
