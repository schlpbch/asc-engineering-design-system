# ModalHeader

The top section of a modal dialog containing a title, optional subtitle, and an × close button. Can also be used standalone.

## Accessibility

- The title is exposed as the accessible name of the dialog via `aria-labelledby`.
- The × button uses `aria-label="Close"`.
