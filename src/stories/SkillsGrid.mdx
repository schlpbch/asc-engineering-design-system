# SkillsGrid

A categorized grid displaying technical skills with proficiency levels. Skills are grouped by category (e.g. Languages, Infrastructure, Frontend).

## Skills data shape

```ts
type Skill = {
  name: string;
  level: 'expert' | 'proficient' | 'familiar';
};

// Pass as a record of category name → skills
skills: Record<string, Skill[]>
```

## Accessibility

- Section heading renders as `<h2>`; category headings as `<h3>`.
- Proficiency levels are rendered as visible text alongside the skill name.
