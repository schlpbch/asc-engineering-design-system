# Avatar

Represents a user or entity with an image or initials fallback. Supports an optional presence status indicator.

## Usage

<Canvas of={Stories.WithImage} />
<Controls of={Stories.WithImage} />

## Image vs. initials

Provide `src` + `alt` for an image avatar. If `src` is absent or fails to load, `initials` is shown as a colored placeholder.

## Accessibility

- Always provide `alt` text describing the person or entity.
- Status indicators are decorative by default; include the status in accessible text if it conveys critical meaning.
