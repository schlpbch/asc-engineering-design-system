# ASC Engineering Design System

A component library built with [Astro](https://astro.build) and [Tailwind CSS v4](https://tailwindcss.com), providing 26 primitive components and 8 higher-level patterns.

**Package:** `@asc-engineering/design-system` · **Version:** 0.3.1

---

## Getting started

```bash
pnpm add @asc-engineering/design-system
```

Import components directly:

```astro
---
import Button from '@asc-engineering/design-system/components/Button.astro';
import Alert from '@asc-engineering/design-system/components/Alert.astro';
---

<Button variant="solid" intent="primary">Click me</Button>
```

---

## Design tokens

| Token | Values |
|-------|--------|
| **Intent** | `primary` · `secondary` · `success` · `warning` · `error` · `info` |
| **Size** | `xs` · `sm` · `md` · `lg` · `xl` |
| **Variant** | per-component — see individual docs |

The design token palette is rooted in `accent-*` (teal) and `slate-*` (neutral) Tailwind scales, with semantic intent colors mapped to standard hues (green, amber, red, blue).
