# ModalFooter

The bottom bar of a modal dialog containing a dismiss button, optional help text, and an `actions` slot for additional buttons. Can also be used standalone.

## Accessibility

- The close button is associated with the modal via `closeButtonId`/`aria-controls`.
- Destructive actions in the `actions` slot should have an `intent="error"` visual style and descriptive labels.
