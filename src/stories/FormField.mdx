A wrapper pattern that pairs a label with any form control (input, textarea, select, etc.) and provides consistent validation message and hint text rendering.

## Accessibility

- The `<label>` is associated with the field via `for`/`id`.
- Validation messages are linked via `aria-describedby` so screen readers announce them.
- Required indicator (*) is supplemented by `aria-required="true"` on the field.
