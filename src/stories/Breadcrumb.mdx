# Breadcrumb

A navigational aid showing the current page's position within a site hierarchy. The last item is the current page and is rendered without a link.

## Accessibility

- Wrapped in `<nav aria-label="Breadcrumb">` for landmark navigation.
- The current page item has `aria-current="page"`.
- Separator characters are `aria-hidden` to prevent repetitive announcements.
