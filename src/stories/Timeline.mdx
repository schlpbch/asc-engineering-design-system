# Timeline

A vertically stacked list of experience or history entries, each with a title, company, location, period, and bullet points.

## Timeline item data shape

```ts
type TimelineItem = {
  title: string;
  company: string;
  location: string;
  period: string;
  bullets: string[];
};
```

## Accessibility

- Each entry is a list item (`<li>`) inside an ordered list (`<ol>`).
- Dates are wrapped in `<time>` where ISO values are available.
