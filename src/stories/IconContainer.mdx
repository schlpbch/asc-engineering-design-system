# IconContainer

A decorative container that applies a colored background and matching tint to an SVG icon. Use in feature lists, empty states, or card headers to give icons visual hierarchy.

## Usage

## Accessibility

- The icon inside should be `aria-hidden="true"` if the surrounding context already describes the action or concept.
- If the container conveys standalone meaning, add a visually-hidden label nearby.
