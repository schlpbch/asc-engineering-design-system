# FilterButtons

A group of toggle buttons for filtering a list or grid. Supports both single-select (radio-like) and multi-select (checkbox-like) modes.

## Filter data shape

Each filter in the `filters` array takes:

```ts
{
  id: string;       // unique DOM id
  label: string;    // button text
  active: boolean;  // initial active state
}
```

## Accessibility

- Buttons use `aria-pressed` to communicate toggled state to screen readers.
- The group label is exposed via `aria-label` on the containing element.
