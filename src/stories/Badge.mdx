# Badge

A compact label used to display status, category, or count information. Common uses include status chips, filter tags, and notification counts.

## Variants

| Variant | Description |
|---------|-------------|
| `default` | Tinted background with matching text |
| `outline` | Border only, no background fill |
| `filled` | Solid background for high-emphasis labels |
| `dot` | Prefixes a colored dot indicator |

## Accessibility

- The remove button includes `aria-label="Remove"` for screen reader users.
- Status badges should supplement — not replace — meaningful page context.
